Contains metabolic and hypoxia tolerance (Pcrit) data collected on Atlantic cod </i>Gadus morhua</i> at Kristineberg Marine Station in June-July, 2025. 
All data can be found in the excel file pcrit_tidydata.xlsx. See the readme sheet on this file for definitions of all variables.
The word document Cod Pcrit Methods contains a description of the methods used to collect and analyse data, as well as several key figures from the analysis with captions.
The R script Atlantic Cod Pcrit.R has all of the code used to calculate metabolic rates and Pcrit, as well as all subsequent analysis and figure generation.
